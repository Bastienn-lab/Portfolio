Design of a Secure Computer Network

As part of SAE 33, we designed a network for a multi-site organization, structured into several VLANs (Employees, Administration, Telephony, WiFi, Management) with an appropriate IP addressing plan. We deployed DHCP, configured Spanning-Tree to prevent loops, implemented HSRP for gateway high availability, and used OSPF for dynamic routing between the different segments.

On the security side, a Cisco ASA firewall was configured to filter inbound and outbound traffic, along with an SSL VPN AnyConnect for secure remote access. We also implemented advanced security measures: BPDU guard, DHCP snooping, ARP inspection, and SSH on all critical devices.

Tools Used

  • GNS3
  • Wireshark
  • Cisco ASA 5505
