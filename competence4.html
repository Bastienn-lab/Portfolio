Checklist application

During the Checklist project in collaboration with Thales Alenia Space, I developed a web application hosted on a Raspberry Pi aimed at improving team efficiency in designing and securing a project. This application allows creating, managing, and sharing best practices with different user permission levels.

The site integrates database management and security features, using technologies such as PHP, HTML, CSS, and JavaScript. Users can view, export, and manipulate best practices through an intuitive interface.

This project allowed me to strengthen my skills in database management, web development, and IT security.
