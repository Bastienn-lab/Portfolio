Creating a computer network for a small business

As part of my studies, I developed a computer network for an SME, using tools like GNS3 and Packet Tracer to simulate and demonstrate the implementation of essential services, including routing, switching, and security with a Cisco ASA 5505 firewall.

This project allowed me to practice technical networking skills and strengthen my expertise in cybersecurity solutions tailored for small businesses.

Tools Used

  • Cisco Packet Tracer
  • Wireshark
  • GNS3
  • Enregistreur d'écran

Languages Used

  • TCL
