TraceCord – Discord message analysis application

As part of SAE 302, I participated in the development of TraceCord, an Android application connected to a PHP/MySQL server, allowing the display of Discord user messages and analyzing their toxicity.

The project relies on a Python bot to extract messages, a PHP REST API to access the database, and a mobile interface developed in Java using Android Studio.

This project allowed me to deepen my skills in distributed web architecture, server deployment (Apache/MariaDB), and inter-service communication via APIs.

Tools Used

  • Android Studio
  • Apache 2
  • MariaDB
  • phpMyAdmin

Languages Used

  • Java
  • Python
  • PHP
  • SQL
