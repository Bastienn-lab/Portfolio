Intrusion test on hybrid infrastructure (Grey box)

As part of SAE 3 Cyber.04, I carried out a complete penetration test on two target machines (vulnerable Samba server and Web server with DVWA), following a grey box approach.

I simulated a realistic attack including network reconnaissance, exploitation (Samba, injection, upload), SSH pivoting, persistence (Cron, VNC), and strategic recommendations, with a structured report delivered to the fictional client Connect3s SAS.

This project allowed me to strengthen my skills in security auditing, vulnerability exploitation (CVE-2017-7494), network pivoting, and professional cybersecurity report writing.

Tools Used

  • Nmap
  • Burp Suite
  • Metasploit
  • SQLMap
  • John The Ripper
  • ProxyChians
